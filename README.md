# CE-ESY-assignments
homework
الفكرة الأساسية: مخزن بحجم ثابت ( 12)، يتم إدخال البيانات فيه بشكل دائري (عند الوصول للنهاية يرجع للبداية).
المكونات:
buffer: مصفوفة تخزن الحروف
head: مؤشر القراءة
tail: مؤشر الكتابة
count: عدد العناصر داخل المخزن
الدوال:
init تهيّئ القيم
isFull و isEmpty للتحقق من الحالة
write تضيف حرف إذا فيه مساحة، وإلا تطبع Overflow
read ترجع حرف إذا فيه بيانات، وإلا Underflow
في main:
يتم دمج "hadeel" مع "CE-ESY" → يعطي hadeelCE-ESY
